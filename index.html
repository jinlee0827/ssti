2. 평소 땀은 어느 정도로 흘립니까?

3. 추위와 더위 중 어떤 것을 더 힘들어합니까?

4. 목소리 톤이나 크기는 어떻습니까?

5. 걸음걸이는 어떤 편입니까?

6. 새로운 일을 시작할 때 어떤 편입니까?

7. 대인관계에서 주로 어떤 역할을 합니까?

8. 스트레스를 받을 때 주로 어떻게 반응합니까?

9. 평소 감정 표현은 어떤 편입니까?

10. 꼼꼼함과 관련하여 자신은 어떤 편입니까?

11. 평소 소화력은 어떻습니까?

체형 사진 업로드 (선택 사항)

사진을 첨부하시면 더 정확한 분석에 도움이 될 수 있습니다. (현재 기능 구현 안됨)

🌞태양인 (리더형 / 에너지 과잉형)

특징

  • 상체 발달, 활동적, 자존감 높음, 간 기능 왕성

추천 음식

  • 보리밥, 도토리묵, 수박, 배, 미역, 우엉
  • 담백하고 차가운 성질의 음식

피해야 할 음식

  • 돼지고기, 치즈, 튀김류, 밀가루, 기름진 음식

건강 습관

  • 과격한 운동보다는 유산소 중심
  • 식욕 절제 및 과식 주의
  • 스트레스 조절 중요

🌳태음인 (안정형 / 체력우세형)

특징

  • 하체 발달, 체격 큼, 느긋함, 폐 기능 약함

추천 음식

  • 녹두, 율무, 도라지, 무, 생강차, 나물류
  • 소화 돕는 따뜻한 음식 위주

피해야 할 음식

  • 기름진 음식, 고지방 육류, 밀가루

건강 습관

  • 규칙적인 운동 (등산, 걷기 등)
  • 땀 배출을 통한 노폐물 제거
  • 과음 및 과식 주의

🔥소양인 (외향형 / 추진력 있는 스타일)

특징

  • 상체 발달, 열 많음, 급한 성격, 비위 기능 강함

추천 음식

  • 오이, 배추, 메밀, 가지, 연근, 국수
  • 시원하고 담백한 음식

피해야 할 음식

  • 닭고기, 술, 커피, 인삼, 삼계탕

건강 습관

  • 무리한 일이나 과로 피하기
  • 과도한 카페인, 열성 식품 자제
  • 감정 기복 관리 필요

❄️소음인 (내향형 / 섬세하고 민감한 스타일)

특징

  • 소화기 약함, 추위 탐, 예민함, 신경성 질환 많음

추천 음식

  • 생강, 마늘, 대추, 양배추, 찹쌀죽, 곰탕
  • 따뜻하고 부드러운 음식

피해야 할 음식

  • 아이스크림, 찬 음식, 날 음식, 커피

건강 습관

  • 소화 잘되는 식사, 규칙적인 수면
  • 차가운 환경 피하고 따뜻하게 유지
  • 지나친 걱정이나 불안 줄이기
